## Project Description

Quick-Calc is a simple calculator application developed in Python.  
It supports the following operations:

- Addition
- Subtraction
- Multiplication
- Division (with zero-division handling)
- Clear function (reset to 0)

The main goal of this project is to demonstrate software testing practices including unit testing and integration testing using pytest.

## Running Tests

To run the test suite:

pytest

All tests should pass successfully.

---

## Testing Framework

This project uses pytest as the testing framework.

Pytest is a lightweight and powerful Python testing tool.
It allows writing simple test functions and provides clear test reports.
It is widely used for unit and integration testing in Python projects.
